# ADR-001: Tech Stack Selection

## Status
Accepted

## Context
选择项目技术栈。

## Decision
- Framework: Next.js 15 (App Router)
- Language: TypeScript
- Database: PostgreSQL + Drizzle ORM
- Auth: NextAuth.js
- Styling: Tailwind CSS + shadcn/ui
- Testing: Vitest + Playwright
