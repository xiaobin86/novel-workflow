# PRD — 删除项目功能

> **版本**：v1.0  
> **日期**：2026-04-20  
> **状态**：已实现（2026-04-20）  

---

## 1. 背景与目标

当前项目列表页只支持创建项目，无法删除。用户在测试或废弃某个小说项目后，数据和文件会一直残留在磁盘上，无法清理。

**目标**：在项目列表页和项目详情页提供删除入口，彻底清除一个项目的所有数据。

---

## 2. 用户故事

> 作为用户，我希望能删除一个不再需要的小说项目，包括其分镜、图片、音频、视频和状态文件，使磁盘空间得到释放，且该项目从列表中消失。

---

## 3. 功能范围

### 3.1 删除入口

| 位置 | 形式 |
|------|------|
| 项目列表页（`/projects`）| 每个项目卡片右上角「⋯」菜单或直接显示删除图标 |
| 项目详情页（`/projects/[id]`）| Header 右侧「删除项目」按钮 |

### 3.2 删除行为

1. 点击删除按钮 → 弹出**确认对话框**，显示项目名称和警告文案
2. 用户点击「确认删除」→ 调用后端 API 执行删除
3. 删除成功 → 跳转到 `/projects` 列表页

### 3.3 删除范围

删除以下内容（全部不可恢复）：

| 内容 | 路径 |
|------|------|
| 项目状态文件 | `{PROJECTS_BASE_DIR}/{id}/state.json` |
| 分镜文件 | `{PROJECTS_BASE_DIR}/{id}/storyboard.json` |
| 输入文本 | `{PROJECTS_BASE_DIR}/{id}/input.txt` |
| 生成图片 | `{PROJECTS_BASE_DIR}/{id}/images/` |
| TTS 音频 | `{PROJECTS_BASE_DIR}/{id}/audio/` |
| 视频片段 | `{PROJECTS_BASE_DIR}/{id}/clips/` |
| 最终视频 | `{PROJECTS_BASE_DIR}/{id}/output/` |
| 整个项目目录 | `{PROJECTS_BASE_DIR}/{id}/` |

### 3.4 安全限制

- **正在执行中的步骤不允许删除**：如果项目有任何步骤状态为 `in_progress` 或 `paused`，显示错误提示，要求用户先停止任务
- 删除操作**不可撤销**，确认弹窗中明确标注

---

## 4. API 设计

### DELETE `/api/projects/[id]`

**请求**：无 Body

**处理逻辑**：
1. 读取 state.json，检查是否有 `in_progress` / `paused` 步骤
2. 如有活跃步骤，返回 `409 Conflict`
3. 递归删除 `{PROJECTS_BASE_DIR}/{id}/` 目录
4. 返回 `204 No Content`

**错误响应**：

| 状态码 | 原因 |
|--------|------|
| 404 | 项目不存在 |
| 409 | 有步骤正在执行中 |
| 500 | 文件系统错误 |

## 5. UI 设计

### 确认对话框

```
┌─────────────────────────────────────────┐
│  删除项目                                │
│                                         │
│  确定要删除「青冥秘境·剑影寒潭」吗？      │
│                                         │
│  此操作将删除所有生成内容，包括图片、     │
│  音频、视频和分镜文件，且不可恢复。       │
│                                         │
│          [ 取消 ]  [ 确认删除 ]          │
└─────────────────────────────────────────┘
```

- 「确认删除」按钮使用红色 destructive 样式
- 项目名称加粗显示

### 项目列表卡片

在卡片 hover 时右上角显示删除图标（垃圾桶），点击触发确认弹窗。

---

## 6. 非功能需求

- 删除操作完成时间 < 5s（本地磁盘）
- 删除失败时给出明确错误信息，不静默失败

---

## 7. 不在范围内

- 批量删除（v1.0 不做）
- 回收站 / 软删除（v1.0 不做）
- 仅删除部分步骤产物（v1.0 不做）
